# SiembrasMiantioquia_WinForms
Aplicaciones tipo Desktop para demostrar conectividad con SQLite.

- Framework: .NET 4.8.x
- Tipo proyecto: Windows Forms
- ORM: Dapper
- DB: Sqlite 3.x
